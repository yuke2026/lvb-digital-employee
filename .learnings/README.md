# .learnings/

项目级学习记录，跨 Agent 共享（Hermes Agent、Claude Code、Codex 等均可读写）。

## 文件说明

| 文件 | 用途 |
|------|------|
| `LEARNINGS.md` | 纠正记录、最佳实践、知识缺口 |
| `ERRORS.md` | 技术踩坑 + 解决方案 |
| `FEATURE_REQUESTS.md` | 功能需求记录 |
| `sessions/` | 每次复杂会话的完整归档 |

## 规范

- 使用 `grep` 搜索已有条目，避免重复
- 关联条目用 `See Also` 链接
- 重要条目逐渐升级到 Skill 或 Memory
