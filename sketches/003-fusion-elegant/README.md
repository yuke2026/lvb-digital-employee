# 003 融合版 - 设计说明

## 融合策略

| 维度 | 采用 | 来源 |
|------|------|------|
| 色系 | 深色基调 #0c0f18 + 蓝色强调 #3b82f6 | 001 主体 + 002 精确色值 |
| 字体 | Inter + 略大字号(14px) | 002 |
| 布局 | 三栏：侧边(224px) + 对话(弹性) + 任务面板(320px) | 001 |
| 专家选择器 | 横向卡片、圆形头像、Notion 式简洁 | 002 |
| 自动化面板 | 保留密集任务列表 + 卡片 hover 上浮 | 001 + 002 |
| 快捷操作 | Pill 样式按钮 + hover 效果 | 002 |
| 输入框 | 胶囊形 + 圆形发送按钮，Enter 发送 | 002 |
| 右侧面板 | Tab 切换：自动化 / 报告，两个视图 | 新增 |
| 交互 | Toggle 开关、slideUp 动画、脉冲在线状态 | 001 |
| 数字员工列表 | 圆形头像徽章替代文字图标 | 002 |

## 新增交互

- **右侧面板 Tab 切换**：自动化任务 / 战略报告两个视图，点标签切换
- **Pill 快捷按钮**：点击自动填入输入框，支持 4 种快捷指令
- **Enter 发送**：输入框按 Enter 直接发送，Shift+Enter 换行
- **报告卡片列表**：hover 上浮效果
